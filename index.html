Getting started with Python (Beginner’s Tutorial)

2022-05-19 — 2 min read

Getting Started #

Hi, In this tutorial you will how to get started with Python. You will go through python installation and some basics of python. But firstly, what is python. Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically-typed and garbage-collected.

  • Dynamically Typed - These are languages which assign the data type to a variable at runtime based on their value. Eg - Python, Ruby & etc. Read More.

  • Garbage Collected - It is a form of automatic memory management. It frees the memory which was allocated by the program.

Pros #

The pros of Python are -

  • Easy to Read, Learn and Write

  • Improved Productivity

  • Dynamically Typed

  • Free and Open-Source

  • Vast Libraries Support

Cons #

The cons of python are -

  • Slow Speed

  • Not Memory Efficient

  • Weak in Mobile Computing
